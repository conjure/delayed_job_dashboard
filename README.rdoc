= Delayed Job Dashboard

== Installation

Include in your Gemfile

    gem 'dj_dashboard', git: 'git://github.com/conjure/delayed_job_dashboard.git'

Update your bundle

    bundle install

Mount in your routes.rb

    mount DjDashboard::Engine, at: "djdashboard"
